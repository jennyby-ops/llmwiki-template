# 크로스커팅 미결사항 (Cross-Cutting Issues)

> **목적**: 2개 이상의 주제에 걸친 미결사항을 여기서 통합 관리.
> 각 주제 세션은 이 파일을 읽고 → 자기 주제와 연관된 항목 확인 → 결정 시 여기에 상태 업데이트.
> 관리: {담당자} | 최초 생성: {YYYY.MM.DD}

---

## ⚠️ 세션 간 사용 규칙

1. **세션 시작 시**: 이 파일을 `wiki/index.md`, `wiki/log.md`와 함께 반드시 읽는다.
2. **새 크로스커팅 이슈 발견 시**: 이 파일에 X번호로 추가 + 각 주제 issues.md에 "→ cross_issues.md X번 참조" 메모.
3. **결정/해결 시**: 이 파일 상태 업데이트 + 연관 주제 issues.md도 함께 업데이트 + log.md 기록.
4. **issues.md에서 이 파일로 승격 기준**: "다른 주제의 결정에 영향을 받거나 준다"면 크로스커팅.
